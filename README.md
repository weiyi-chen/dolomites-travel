## 关于这份路线

这是一份为期5天的多洛米蒂自驾行程，4人出行，从伦敦出发。核心亮点是**8月12日英仙座流星雨**，以Misurina露营地（海拔1756m）为大本营，东西两区徒步全覆盖。

```
伦敦 ✈️ 威尼斯/米兰 → 奥蒂塞伊（西部1晚）→ Misurina露营地（东部3晚）→ 返程
```

---

## 行程亮点

| 日期 | 地点 | 亮点 |
|------|------|------|
| Day 1 | 奥蒂塞伊 | 抵达·适应高度 |

| Day 3 🌠 | Braies湖 + Cadini观景台 | 清晨划船 + 深夜流星雨 |

| Day 5 | Cinque Torri + 返程 | WW1战壕遗址 + 360°全景 |

## 进出方案

**方案A（推荐）：威尼斯进 + 威尼斯出**
- 伦敦直飞VCE · Ryanair / EasyJet 均有
- 到奥蒂塞伊约2.5小时
- 同机场还车，无异地附加费

**方案B：米兰进 + 威尼斯出**
- 票价有时更低，但需支付异地还车费
- 到奥蒂塞伊约3.5小时
- 综合比价后决定
